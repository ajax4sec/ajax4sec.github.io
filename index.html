Research:

My research interests focus on software security and software engineering, including program analysis, vulnerability analysis and malware detection .
The methods involves taint analysis, symbolic execution, fuzzing, virtualization and reverse engineering.
I worked on several projects, including

  • Fuzzing: Improve fuzzing strategies and extend fuzzing framework to kinds of targets.
  • Reassemble disassembling: Improve Project Uroboros which is a binary analysis and rewrite tool and apply it to kinds of scenarios.
  • Heap overflow vulnerability discovery/detection/assessment: Designed and implemented a system to discover/detect/assess heap overflow vulnerabilities based on dynamic taint analysis and symbolic execution.
  • Application-oriented taint analysis system development: Implemented hook functions for taint sources and heap allocation operations.
  • Malware detection and analysis platform development: Implemented interaction analysis tools for malware detection based MFC.
